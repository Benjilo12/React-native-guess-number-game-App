A fun and interactive number guessing game built with React Native and Expo, following a complete course on building mobile apps with React Native.

Players try to guess a randomly generated number, and the app provides feedback until the correct number is guessed.


## Features

- Random number generation
- User-friendly input for guesses
- Tracks number of attempts
- Restart the game anytime
- Built using React Native and Expo

## Prerequisites

Node.js installed

Expo CLI installed (npm install -g expo-cli)


## Run Locally

Clone the project

```bash
  git clone https://github.com/Benjilo12/React-native-guess-number-game-App.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  start
```

Scan the QR code with Expo Go on your phone or run in an emulator.
